##  Overview
The Arte Povera frontend is a high-performance, responsive web application built with **React** and **TypeScript**. It serves as a professional gateway for artists to showcase their work, interact with businesses, and manage their creative portfolios in a streamlined, modern environment.

### Key Features
* **Artist Portfolio System:** Dynamic display of images, videos, and PDF documents using **Cloudinary** for optimized media delivery.
* **Real-Time Communication:** Instant messaging and notification system powered by **Socket.io-client**.
* **Internationalization (i18n):** Multi-language support to ensure accessibility for a global audience of artists and recruiters.
* **Responsive Design:** A mobile-first UI built with **Tailwind CSS** to ensure a seamless experience across all devices.
* **Type Safety:** 100% TypeScript implementation for robust state management and component reliability.

##  Tech Stack
* **Framework:** React 18+
* **Build Tool:** Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Routing:** React Router
* **Real-Time:** Socket.io-client
* **Deployment:** Vercel
